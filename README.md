# rogue-scholar-bert

Rogue Scholar BERT is an API service for classifying scholarly texts using a fine-tuned BERT model from Hugging Face:

https://huggingface.co/OpenAlex/bert-base-multilingual-cased-finetuned-openalex-topic-classification-title-abstract

The model was developed by OpenAlex and CWTS.
